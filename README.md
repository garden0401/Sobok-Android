## 서비스 설명
` 소복소복` : 소중한 사람과 함께하는 복약 체크 서비스

`비전` : 고객들이 약을 제 때 약을 잘 챙겨 먹어서, 건강해질 수 있도록 한다.

`핵심가치` 

- 간편한 것 (쉽고, 약 관리만 깔끔하고 간편하게)
- 관계 (따뜻함, 상호작용, 챙김 받는 느낌)
- 소복소복이 복약 확인을 대신 해준다. (딸의 확인하기 귀찮음 문제를 해결한다.)
- 실제로 약을 먹을 수 있도록 하는 것.
